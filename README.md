# egg-pkg-template

一个使用 [pkg](https://github.com/vercel/pkg) 将 [Egg.js](https://eggjs.org) 后端 Node 服务打包为可执行程序的样例工程。
更多内容请参见：[掘金 | Egg.js + pkg 打包可执行程序踩坑笔记](https://juejin.im/post/5f2051fb6fb9a07e8151e168)

## Quick Start

```bash
$ npm install -g pkg
$ npm install 
$ npm run build
$ open http://localhost:7001/
```
